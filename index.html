A.E.O.L.U.S Flight Controller

Adaptable & Extensible Open-Loop Unmanned System

Built from the ground up, this quadrotor firmware was designed using a Model-Based Design (MBD) approach in MATLAB Simulink. Instead of using off-the-shelf PID tuners, I derived the rigid-body physics equations to create a high-fidelity continuous-time plant model.

Key Technical Implementations:

  • Mathematical Modeling: Formulated a nonlinear plant using a 4x4 control allocation matrix inverse (A⁻¹) to map desired torques (roll, pitch, yaw) and total thrust to individual motor speeds.
  • Sensor Fusion: Developed a custom complementary filter (α=0.9995) to seamlessly fuse high-frequency gyroscope data with stable accelerometer and tilt-compensated magnetometer readings, eliminating drift.
  • Advanced Control Laws: Cascaded attitude-rate PID controllers augmented with tilt-compensated thrust equations, ensuring that the quadrotor maintains its vertical altitude setpoint even during aggressive pitch/roll maneuvers.

Autonomous Power Grid Control via DRL

A sophisticated reinforcement learning framework designed to prevent cascading failures in power distribution networks by dynamically controlling demand scaling and transmission line topology.

System Architecture & Innovations:

  • Custom Physics Engine: Engineered a highly optimized C++ physics simulator to model power flows across a 100-substation grid, complete with a real-time Raylib visualizer.
  • Inter-Process Communication: Bridged the C++ environment with a Python/PyTorch training loop using ultra-fast ZeroMQ (REQ/REP) sockets, bypassing standard Python bottlenecks.
  • Soft Actor-Critic (SAC) Agent: Trained a Twin-Critic SAC agent capable of continuous action spaces. Features auto-tuned entropy (τ=0.005) and a heavily engineered 7-component shaped reward function that balances demand service, overload prevention, and topology stability over 8,760-step (annual) episodes.

Pluto Nano Drone Firmware

Developed for the Inter IIT Tech Meet 14.0 (Drona Aviation Problem Statement), focusing on advanced state estimation for micro-UAVs in GPS-denied environments.

Core Contributions:

  • High-Speed Estimation: Architected a dual-loop state estimator running on Cleanflight firmware with a blistering 250Hz main control loop.
  • Multi-Sensor Fusion: Fused data from IMU, optical flow sensors, Time-of-Flight (ToF) sensors, and barometers. Deployed a mathematically rigorous 3rd-order complementary filter for drift-free altitude, velocity, and bias correction.
  • Drift Suppression: Integrated Zero-Velocity Updates (ZUPT) and dynamically weighted optical-flow/IMU blending based on sensor quality. Designed a cascaded Position → Velocity → Attitude PID loop paired with DCM-based attitude estimation.
